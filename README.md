# Holberton Higher Level Programming

Module Holberton School sur la programmation de haut niveau en Python : accès aux bases de données, sérialisation et services web.

## Projets

| Dossier | Contenu |
|---|---|
| `python-object_relational_mapping` | Lien entre Python et MySQL, requêtes via MySQLdb puis ORM SQLAlchemy |
| `python-serialization` | Sérialisation et désérialisation de données (JSON, pickle, CSV, XML) |
| `restful-api` | Consommation et création d'API REST avec Flask |

## Prérequis

- Python 3.8 ou plus récent
- MySQL pour la partie ORM
- Flask et SQLAlchemy selon les projets

## Utilisation

```bash
cd restful-api
python3 nom_du_script.py
```
